Example
{: .label .label-blue }

<p align = "justify">
  <i>Determine the IDs of Degrees of Freedom in a frame structure with ten nodes.</i>
</p>

```python
# Call function
DOF_GLOBALNODAL = DOF_GLOBAL_ASSEMBLY(TYPE_ELEMENT = 0, N_DOFSNODE = 3, N_NODES = 10)

# result
print("nodal dofs: ", "\n", DOF_GLOBALNODAL)
```
